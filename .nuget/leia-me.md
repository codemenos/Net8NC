# *Leia-me*

### Use o comando a seguir para instalar a ferramenta de linha de comando do Entity Framework Core:

```bash
dotnet tool update dotnet-ef --global
```

É esperado que utilize a versão 8.0.1 ou superior.
Caso não tenha a ferramenta instalada, utilize o comando a seguir:

```bash
dotnet tool install --global dotnet-ef
```
